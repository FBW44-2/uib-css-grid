# CSS Grid

## The Grid Container

- grid-template-columns
- grid-template-rows

fr - fraction of free space

## The Grid Items

- grid-column
- grid-row

## Centering with grid

```css
.grid {
    display: grid;
    justify-content: center;
    align-items: center;
}
```

Make sure the container has some height, otherwise you want see the vertical centering
